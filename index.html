Who is Ishant?

I'm a CS Engineer from VIT Chennai, specialising in Information Security — which means I learned to build systems and then learned how to break them (ethically).

My stack is wide on purpose. I write ML models that detect ransomware, build full-stack platforms with MERN, engineer Generative AI pipelines with LangChain and RAG, and deploy on AWS. The most interesting problems live at intersections of disciplines.

Currently obsessed with agentic AI workflows — AI systems that perceive, reason, act and learn in loops. Also obsessed with behavioural threat detection and why ML catches malware better than any signature database.

04 — Projects

Things I've Built

01
Cybersecurity · ML · Python · Capstone

AI-Based Ransomware Early Detection System

A proactive endpoint security system that intercepts ransomware before it encrypts a single byte. Watches Sysmon behavioural logs — process trees, file I/O patterns, memory anomalies — and feeds them into a multi-model ensemble (Random Forest + XGBoost + Isolation Forest). Threats flagged in real time, not in the post-mortem report after damage is done. Seven-module detection pipeline. Designed to be production-deployable on Windows endpoints.

Where deep learning meets diagnostic medicine.
